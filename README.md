## ✨ Features

- **Golden Particle Effects** — Floating magical sparkles across the entire UI
- **Castle Silhouette** — Dark atmospheric castle background with moonlit night sky
- **Wizarding Fonts** — Cinzel Decorative, Cinzel, Crimson Text & MedievalSharp
- **Dark Fantasy Palette** — Deep midnight, gold, crimson, emerald & parchment tones
- **Golden Snitch Icons** — Point values displayed with snitch icons
- **Fully Responsive** — Works seamlessly across all screen sizes

| Emerald | `#1a6b3c` | Solved challenges, success |

## 🚀 Installation

1. Open your Docker container's terminal and run:

   ```bash
   git clone https://github.com/jagdishtripathy/hogwarts.git /opt/CTFd/CTFd/themes/hogwarts
   ```

2. Log in as **Admin** and navigate to:

   > **Admin Panel → Config → Themes**

3. Select **hogwarts** from the theme dropdown and click **Update**.

You're all set! 🎉
